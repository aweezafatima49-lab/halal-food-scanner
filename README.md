AI Halal Food Scanner
Overview:
A smart AI-driven application designed to help consumers instantly verify the Halal status of food products. By scanning ingredient lists, the application uses machine learning to classify and provide clear insights, ensuring users make informed dietary choices.
Key Features:
AI-Powered Scanning: Efficiently processes ingredient text to identify potential non-Halal components.
Real-time Analysis: Provides instant results via an intuitive user interface.
Easy Accessibility: Built using Streamlit for a fast and responsive web-based experience.
Tech Stack:
Language: Python
Framework: Streamlit

How to Use:
Visit the live app link provided above.
Input the ingredient list into the text area.
Click on the scan button to view the results instantly.
